Verizon Rapid Deliver Mobile (Sales 2 Go)

Web application for Verizon sales representatives and technicians to be able to manage contracts for the Verizon Rapid Delivery system from tablet or mobile.

This project is a web application for in-the-field Verizon sales representatives and technicians to be able to manage contracts for the Verizon Rapid Delivery system from tablet or mobile. The front-end for this project leverages the most cutting edge trends in JavaScript, built with ReactJS and Redux.

My biggest role on the project was in the design and implemention of a feature called EZStatus, which allows the user to check on the status of Order Requests and Service Orders. In addition to developing the UI for EZStatus, I coordinated with and delagated to several teams for the development of the APIs. I also worked on the implementation of the Push notifications system.

  • Conceptual Design and Approach Documentation
  • Preliminary Design Documentation
  • Implementation
  • Unit Testing

VetCheq Cloud

VetCheq Cloud provides a window into your horse's health data and lets you be with your horse no matter where you are

At PonyUp Technologies, I developed the VetCheq Cloud single page application, which allows users to view the health data collected by their VetCheq remote equine health monitoring device. I was responsible for parts of the full development process, including UX design and prototypes, software architecture design, and coding. During this project, I redesigned the company's previous software architecture in favor of more modern practices using an MVC structure, a RESTful Ruby on Rails API, and a responsive AngularJS based single page front end client.

VetCheq Application

A powerful non-invasive Equine health diagnostic tool that brings your horse's health to your fingertips wherever you are

This project is a Java-based application that is meant to stream live equine health data from the VetCheq bluetooth device. I was responsible for the UX design and development of the JavaFX user interface. It communicates via TCIP sockets with a program that interfaces directly with the VetCheq device, allowing veterinarians and other equine professionals to trigger recordings to be taken remotely.

Process

  • User Research
  • Wireframes and UX Design
  • Requirement Gathering and Analysis
  • Software Architecture Design
  • Code Implementation

Tools

  • JavaFX
